# 🗺️ AI Agent 학습 지원 웹페이지 UI/UX 설계 (Play & Explore Theme)

## 1. 디자인 시스템 정의 (Visual System)

브랜드 정체성인 '놀이'와 '탐험' 감성을 극대화하기 위해 감정(Emotion)과 색상(Color)을 직접 연결하는 팔레트를 정의합니다.

### 1.1. 컬러 팔레트 (Color Palette)
| 역할 | 색상 이름 | HEX 코드 | 용도 및 감성 |
| :--- | :--- | :--- | :--- |
| **Primary (탐험)** | Deep Ocean Blue | `#1E40AF` | 신뢰감, 깊이 있는 탐험, 안정감 |
| **Secondary (놀이)** | Sunny Yellow | `#FFD700` | 활력, 긍정, 즐거움 (강조 색상) |
| **Accent (감정)** | Coral Pink | `#FF6B6B` | 감정 표현, 따뜻함, 상호작용 유도 |
| **Neutral (배경)** | Soft Cream | `#FFFBF5` | 부드러운 배경, 학습 공간 제공 |
| **Tertiary (보물)** | Forest Green | `#4CAF50` | 성장, 자연, 발견 (보상) |

### 1.2. 타이포그래피 (Typography)
*   **Font Family:** Pretendard (친근하고 가독성이 높은 고딕 계열)
*   **Heading (H1, H2):** Poppins (역동성과 명료성 강조)
*   **Body Text:** Pretendard (최대 가독성 확보)
*   **스타일:** 모든 텍스트에 그림자(Soft Shadow)를 적용하여 입체감과 친근함을 부여합니다.

### 1.3. 아이콘 및 요소 스타일
*   **아이콘 스타일:** Line/Outline 기반, 모서리가 둥글게 처리된(Rounded Corners) 친근한 스타일.
*   **상호작용:** 모든 버튼과 카드에는 마우스를 올렸을 때 색상이 부드럽게 변화(Hover Effect)하며, 클릭 시 미세한 애니메이션(Bounce/Pulse)을 추가하여 '놀이' 요소를 강화합니다.

## 2. 주요 화면 와이어프레임 (Wireframe Structure)

### 2.1. 홈 화면 (The Exploration Hub)
**목표:** 사용자에게 즉각적인 몰입감과 다음 탐험으로의 동기를 부여합니다.

| 영역 | 구성 요소 | 상세 디자인 및 기능 | 레이아웃 좌표 (상대적) |
| :--- | :--- | :--- | :--- |
| **A. 헤더 (Header)** | 로고, 사용자 프로필, 설정 버튼 | 로고는 '모험 지도' 모양의 심볼 사용. 프로필은 캐릭터 아바타와 함께 표시. | Top 0% - 8% |
| **B. 메인 액션 (Main Action)** | **오늘의 미션 카드 (Daily Quest Card)** | 가장 큰 영역. 오늘의 학습 목표(예: '기쁨의 색칠 놀이')를 카드 형태로 제시. **Sunny Yellow** 배경으로 강조. | Middle 10% - 45% |
| **C. 추천 탐험 (Recommended)** | **다음 보물찾기 (Next Treasure Hunt)** | 사용자의 진행 상황이나 다음 단계 학습 모듈을 카드 3~4개로 제시. **Deep Ocean Blue** 배경에 **Coral Pink**으로 강조. | Middle 45% - 75% |
| **D. 학습 경로 (Progress Bar)** | **나의 탐험 지도 (My Map)** | 현재까지 완료한 학습 모듈을 지도 형태로 시각화. 완료된 부분은 **Forest Green**으로 채워짐. | Bottom 25% - 100% |
| **E. 네비게이션 (Navigation)** | 하단 탭 바 (Tab Bar) | 홈, 모듈, 내 지도, 설정 4개 탭. 아이콘은 단순하고 직관적인 놀이 요소(돋보기, 색연필, 지도, 설정) 사용. | Bottom 10% - 100% |

### 2.2. 학습 모듈 화면 (The Learning Module View)
**목표:** 실제 학습 활동에 완전히 몰입할 수 있는 환경을 제공합니다.

| 영역 | 구성 요소 | 상세 디자인 및 기능 | 레이아웃 좌표 (상대적) |
| :--- | :--- | :--- | :--- |
| **A. 모듈 제목** | 현재 학습 주제 표시 | "감정의 색칠 놀이" 등 명확하게 표시. | Top 5% - 15% |
| **B. 학습 활동 영역** | **인터랙티브 활동 캔버스** | **핵심 영역.** 학습 콘텐츠(색칠 도안, 퍼즐 조각, 감정 카드)가 배치되는 영역. 배경은 **Soft Cream**. | Middle 20% - 80% |
| **C. 도구 모음 (Toolbox)** | **도구 선택 패널** | 학습에 필요한 도구(색상 팔레트, 붓, 도구 선택 버튼)를 화면 측면에 고정. **Sunny Yellow**로 버튼 강조. | Left 0% - 15% |
| **D. 피드백 및 보상** | **완료 애니메이션** | 학습 완료 시, 화려하고 긍정적인 애니메이션(폭죽, 반짝임)과 함께 **Forest Green** 보상 메시지 표시. | Bottom 5% - 100% |
