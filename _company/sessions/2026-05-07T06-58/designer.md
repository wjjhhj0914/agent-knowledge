# 🗺️ iOS 학습 지원 에이전트: UI/UX 화면 설계서 (Markdown)

이 설계서는 '놀이'와 '탐험' 감성을 극대화하여 초등학교 저학년 사용자의 몰입도와 학습 동기를 높이는 데 중점을 둡니다.

## 🎯 프로젝트 목표
초등학교 저학년 사용자가 시각적 피드백과 상호작용을 통해 언어 학습에 몰입하고, 감정 학습을 경험하도록 돕는 직관적이고 친근한 학습 환경 제공.

## 🎨 브랜드 가이드라인 적용 (Visual System)

### 1. 컬러 팔레트 (Color Palette)
놀이와 탐험의 감성을 반영하여 따뜻하고 활기찬 파스텔 톤을 사용합니다.

| 역할 | 색상 이름 | HEX 코드 | 용도 |
| :--- | :--- | :--- | :--- |
| **Primary (탐험)** | 민트 그린 | `#98FB98` | 주 탐험 경로, 긍정적 피드백 |
| **Secondary (감정)** | 스카이 블루 | `#87CEEB` | 감정 학습 모듈 배경, 안정감 |
| **Accent (보물)** | 옐로우 골드 | `#FFD700` | 보상, 핵심 액션 버튼, 하이라이트 |
| **Neutral (배경)** | 라이트 베이지 | `#FFF8E1` | 전체 배경, 종이 질감 느낌 |
| **Action (상호작용)** | 코랄 오렌지 | `#FF7F50` | 사용자의 선택, 강조 피드백 |

### 2. 타이포그래피 (Typography)
친근함과 읽기 쉬움을 위해 둥글고 부드러운 산세리프체를 사용합니다.

*   **Font Family:** **Circular Std** 또는 **Nunito Sans** (둥글고 친근한 느낌 강조)
*   **Heading (제목):** Bold, 크기 28pt 이상. 곡선 처리(Rounded corners) 적용.
*   **Body Text (본문):** Regular, 크기 16pt 이상. 충분한 줄 간격(Line Height) 확보.

### 3. UI 컴포넌트 스타일
*   **모서리:** 모든 카드, 버튼, 입력 필드는 **반경 16px 이상**으로 처리하여 부드러운 느낌을 강조합니다.
*   **아이콘:** 플랫하거나 라인 기반의 친근한 일러스트 스타일 (예: 붓, 지도 핀, 보물 상자 모양).
*   **애니메이션:** 모든 상호작용(클릭, 색칠) 시, 부드럽고 느린 이징(Easing) 애니메이션을 적용하여 '놀이'의 느낌을 살립니다.

## 📱 화면 구성 설계 (Screen Flow)

### 1. 메인 대시보드: '모험의 지도' (Home/Map View)
**목표:** 사용자가 오늘 학습할 모듈을 명확하게 제시하고, 다음 활동으로의 동기를 부여.

*   **레이아웃:** 지도(Map)의 느낌을 살려, 배경에 옅은 격자무늬나 구름 패턴을 적용합니다.
*   **중앙 요소:** 현재 진행 중인 학습 모듈(예: '감정의 색칠 놀이')을 **큰 보물 상자 형태의 카드**로 표시합니다.
    *   **카드 디자인:** 배경은 스카이 블루(`\#87CEEB`), 테두리는 코랄 오렌지(`\#FF7F50`).
    *   **텍스트:** 모듈 이름은 큰 폰트로, 진행률은 획득한 보물 개수로 표시.
*   **상단 요소:** 사용자 프로필과 진행 상황 요약 (레벨 표시, 획득한 보물).
*   **하단 요소:** 다음 활동으로 가는 명확한 경로 표시 (예: '다음 보물찾기 시작하기' 버튼).
    *   **버튼 디자인:** 옐로우 골드(`\#FFD700`) 배경에 둥근 모서리를 적용하여 시선을 집중시킵니다.

### 2. 학습 활동 화면: '감정의 색칠 놀이' (Activity View)
**목표:** 시각적 학습 활동에 집중하고, AI의 피드백을 즉각적으로 시각화.

*   **레이아웃:** 학습 도안이 화면의 70% 이상을 차지하도록 중앙에 배치합니다.
*   **학습 영역:** 중앙에 감정(예: 기쁨)을 표현하는 큰 도안(도형 또는 캐릭터 얼굴)이 배치됩니다.
    *   **도안 스타일:** 선이 굵고 명확하며, 색칠 영역은 옅은 격자선으로 미리 표시하여 아이가 어디를 채워야 할지 시각적 안내를 제공합니다.
*   **상호작용:** 사용자가 색칠하거나 드래그할 때, AI가 실시간으로 사용자의 시선/표정 분석 결과를 반영하여 **색상 선택 가이드**를 제공합니다.
    *   **가이드:** 선택 가능한 색상 팔레트(민트, 블루, 옐로우 등)를 화면 측면에 배치하며, 선택된 색상 옆에 간단한 감정 설명(예: "기쁨은 밝은 노란색이야!")을 텍스트로 제공합니다.
*   **피드백:** 올바른 색상 선택 시, 캐릭터가 기뻐하는 **애니메이션(예: 반짝임, 작은 춤)**과 긍정적인 음성 피드백(TTS)을 제공합니다.
*   **도구:** 색상 팔레트와 간단한 도구(붓, 스티커)는 화면 하단에 작고 친근한 아이콘으로 배치합니다.

### 3. 보물찾기 화면: '퍼즐 보물찾기' (Puzzle View)
