# 1. iOS용 학습 지원 에이전트의 UI/UX 화면 설계서 (Markdown)

이 설계서는 초등학교 저학년을 대상으로 하며, '놀이'와 '탐험' 감성을 강조하는 브랜드 톤을 반영하여 시각적 명확성과 학습 몰입도를 극대화하는 데 중점을 둡니다.

```markdown
# 🗺️ iOS 학습 지원 에이전트: UI/UX 화면 설계서 (MVP 기준)

## 🎯 프로젝트 목표
초등학교 저학년 사용자가 시각적 피드백과 상호작용을 통해 언어 학습에 몰입하고, 감정 학습을 경험하도록 돕는 직관적이고 친근한 학습 환경 제공.

## 🎨 브랜드 가이드라인 적용
- **톤앤매너:** 친근함, 놀이, 탐험 (Playful, Exploratory)
- **컬러 팔레트:** 밝고 따뜻한 파스텔 톤 (예: 민트, 연노랑, 하늘색, 따뜻한 주황)
- **타이포그래피:** 둥글고 부드러운 산세리프체 (친근함 강조)
- **핵심 요소:** 모든 인터페이스는 물리적인 '놀이 도구'나 '지도'의 형태로 디자인되어야 함.

## 📱 화면 구성 설계 (Screen Flow)

### 1. 메인 대시보드: '모험의 지도' (Home/Map View)
- **목표:** 사용자가 오늘 학습할 모듈을 명확하게 제시하고, 다음 활동으로의 동기를 부여.
- **UI 요소:**
    - **중앙:** 현재 진행 중인 학습 모듈(예: '감정의 색칠 놀이')을 큰 카드 형태로 표시.
    - **상단:** 사용자 프로필 및 진행 상황 요약 (레벨 표시, 획득한 보물).
    - **하단:** 다음 활동으로 가는 명확한 경로 표시 (예: '다음 보물찾기 시작하기' 버튼).
    - **디자인:** 지도의 느낌을 살려 배경에 옅은 지도 패턴을 적용.

### 2. 학습 활동 화면: '감정의 색칠 놀이' (Activity View)
- **목표:** 시각적 학습 활동에 집중하고, AI의 피드백을 즉각적으로 시각화.
- **UI 요소:**
    - **학습 영역:** 중앙에 감정(예: 기쁨)을 표현하는 큰 도안이 배치됨.
    - **상호작용:** 사용자가 색칠하거나 드래그할 때, AI가 실시간으로 사용자의 시선/표정 분석 결과를 반영하여 색상 선택 가이드 제공.
    - **피드백:** 올바른 색상 선택 시, 캐릭터가 기뻐하는 애니메이션과 긍정적인 음성 피드백 제공.
    - **도구:** 색상 팔레트와 간단한 도구(붓, 스티커)를 친근한 아이콘으로 배치.

### 3. 보물찾기 화면: '퍼즐 보물찾기' (Puzzle View)
- **목표:** 탐색과 문제 해결을 통한 학습을 유도.
- **UI 요소:**
    - **배경:** 보물찾기 맵 또는 미로 형태의 배경.
    - **상호작용:** 화면 내의 숨겨진 단서(이미지)를 찾기 위해 화면을 탐색하도록 유도.
    - **AI 연동:** 특정 단서를 찾았을 때, AI가 관련된 언어 단어(예: '빨간색'에 해당하는 단어)를 제시하고, 이를 맞히면 다음 위치로 이동.
    - **보상:** 보물을 찾을 때마다 시각적 보상(애니메이션, 새로운 스티커) 제공.

### 4. 피드백 및 보상 화면: '탐험 일지' (Journal View)
- **목표:** 학습 결과를 기록하고 성취감을 강화.
- **UI 요소:**
    - **기록:** 오늘 학습한 내용과 달성한 목표를 시각적 기록(사진/그림)과 함께 기록.
    - **감정 기록:** 학습 중 경험한 감정 상태를 간단한 이모티콘이나 색상으로 기록하는 공간.
    - **보상 창:** 획득한 보상(코인, 스티커)을 보여주고, 다음 단계로의 진입 경로를 제시.

## 💻 기술 구현 고려 사항 (Developer Note)
1. **상태 관리:** 모든 학습 단계와 사용자 감정 상태는 Redux 또는 Swift Combine을 사용하여 중앙에서 관리해야 함.
2. **시각적 입력 처리:** 카메라/스크린샷 입력 시, Vision Framework를 통해 실시간으로 얼굴/눈의 움직임을 추적하고 데이터 파이프라인에 입력하는 모듈이 필수적임.
3. **반응성:** 모든 UI 요소는 아이콘과 애니메이션 중심으로 설계하여, 어린 사용자가 텍스트 입력 없이도 즉각적인 시각적 반응을 경험하도록 설계해야 함.
```
